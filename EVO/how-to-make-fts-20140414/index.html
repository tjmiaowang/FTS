FTS Building Block¶

Full-text Search (FTS) building block provides full text search functionality for POI and Name building block. The implementation allows applications to support a convenient one‐shot destination input for POIs and addresses.

What should be read to understand FTS¶

To understand this building block, you need to read the following documents.

  • NDS – Format Specification, 17 Full‐text Search Building Block on page 323
  • NDS – Physical Model Description, FtsDatabase Physical Model
  • SQLite FTS3 and FTS4 Extensions
