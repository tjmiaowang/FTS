==============================
poiFtsAttributeDefinitionTable
==============================
This table contains the definitions of the POI attributes. 

Member detailes
===============

attributeTypeId :
   ID of the POI attribute type.

columnName :
   The column name, e.g. "attributeA", "attributeB".

iconSetId :
   The ID of the associated icon set.
